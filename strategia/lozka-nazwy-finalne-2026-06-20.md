# Łóżka — finalne nazwy kategorii + konsolidacja fraz (anti-kanibalizacja) 2026-06-20

## ZASADA SEO (obowiązuje całą strukturę)
1. Formy fleksyjne / synonimy = JEDNA kategoria (łóżko = łóżeczko = łóżka). Nie dublować.
2. Nazwa kategorii = liczba mnoga, naturalna, ładna. Title nosi główną frazę.
3. Warianty long-tail (kolor, „dla dziewczynki", „z szufladą", rozmiar pojedynczy) = w meta+opisie+H2,
   NIE osobne kategorie — chyba że własny duży wolumen + dość produktów.
4. Jeden koncept = jeden kanoniczny URL. Synonimy konsolidowane w treści.

### NIE robimy (potwierdzone danymi)
- „Łóżka klasyczne" — 0 fraz.
- „Łóżka z grafiką / bajkowe" — fraza tylko 90/mc mimo 79 prod. → filtr merchandisingowy/motywy, nie kategoria SEO.
- „Łóżka montessori" osobno — 30/mc → wariant w opisie „Łóżek dziecięcych".
- Osobne „Łóżeczka" — to forma fleksyjna „Łóżek dziecięcych", konsolidacja w treści (kanibalizacja).

### LUKI (popyt jest, towaru brak → zgłosić klientowi do dosourcingu)
- Łóżka ze zjeżdżalnią — 1 900/mc, 0 prod.
- Łóżka dla chłopca — 1 600/mc, 0 otagowanych (mamy niebieskie/auto, można otagować zamiast dosourcingu).
- Czyste łóżka domki (pojedyncze) — popyt 9 900, tylko 3 szt.

## ANTY-KANIBALIZACJA (title-discipline)
- „Łóżka dziecięce" (głowa) title = „łóżka dziecięce" — NIE „łóżko dziecięce 160x80" (to bierze podstrona 160x80).
- Każda podstrona/facet ma w title swoją frazę + atrybut; głowa zostaje przy frazie head.
- Synonimy (łóżko/łóżeczko, domek/domki) w obrębie JEDNEJ strony (meta description + akapit), nie nowe URL-e.
